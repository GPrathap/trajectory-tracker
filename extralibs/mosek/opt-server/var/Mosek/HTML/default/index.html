Mosek Optimization Server

The MOSEK Optimization Server is a service that solves optimization problem using MOSEK.

The MOSEK Optimization Server can be used from the MOSEK library though the API call "optimizeremote", or by submitting problems directly through the web API.

The server can be set up to allow unencrypted anonymous HTTP access, or to require a login or access token over HTTPS. Optionally, a web API for administering the server can be enabled.

Please see MOSEK documentation for details.
